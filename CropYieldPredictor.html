Cryptocurency Teaching Tool



Creation of a containerized Django front end and a Flask API for the backend

Published on March 2024 - 5 min read

The objective was to develop a website that serves as a teaching material for cryptocurrency for financial economics course at the University of Saskatchewan. The website allows students to mine transactions using the front-end, validate them, and share the mining results with their classmates. Additionally, the professor can post the transaction results to the entire class. To achieve this, I broke down the web application into several steps. However, creating and deploying the application for production level came with many challenges. As a result, I decided to share how I solved some of these challenges.
